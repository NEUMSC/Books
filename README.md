# Books

## 使用方式

### 借书

1. 发一个label为borrow的issue申请，写清楚：找谁借哪本书，归还日期

2. 自己联系此时持有那本书的人

3. 双方达成协议之后，联系管理员更改图书状态信息或者自己pull request

4. close issue

### 还书

1. 由向别人借书的人发一个issue，label为return

2. 当确实归还之后，联系管理员或者自己pull request更改图书状态信息

3. close issue
